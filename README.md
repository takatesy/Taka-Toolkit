# Taka Toolkit

A custom, beginner level offensive security toolkit written in Python from scratch, my first project in python to try and learn.

## Tools

- **Port Scanner** — multithreaded TCP port scanner
- **Banner Grabber** — identifies software and versions running on open ports
- **Directory Brute Forcer** — discovers hidden directories on web servers
- **Password Brute Forcer** — credential attacks against web login forms

## Usage

Run the main toolkit:
```
python toolkit.py
```

## Requirements
```
pip install requests flask
```
